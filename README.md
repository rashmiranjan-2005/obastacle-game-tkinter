# 🎮 Obstacle Game (Tkinter)

> A dynamic obstacle runner game built using Python and Tkinter, showcasing real-time gameplay mechanics, event-driven programming, and GUI-based game development.

---

## 📌 Overview
This project is an interactive obstacle runner game where the player must jump over incoming obstacles to survive.  
The game progressively increases in difficulty, creating an engaging and challenging experience.

---

## ✨ Key Features
- 🕹️ Smooth and responsive jump mechanics  
- 🧱 Two types of obstacles (small & large)  
- 📈 Progressive difficulty scaling  
- 🎯 Real-time score tracking  
- 🔁 Restart functionality after game over  
- ⚡ Lightweight and fast execution  

---

## 🎮 Controls
| Key | Action |
|-----|--------|
| SPACE | Jump |
| ENTER | Restart |

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Library:** Tkinter (GUI)  

---

## 🧠 Concepts Demonstrated
- Event-driven programming  
- Collision detection  
- Game loop implementation  
- Dynamic difficulty adjustment  
- GUI rendering using Tkinter
